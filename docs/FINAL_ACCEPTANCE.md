# 最终验收 v1.1.0

## 范围

按项目计划完成 M1–M12：产品核心闭环、CloudBase、图片与内容安全、关系链、社区治理、运营后台、体验优化和发布工程化。

## 自动化验收

执行：

```bash
npm install
npm run verify
```

验收项：

- 项目结构和页面资源完整
- JSON 文件可解析
- 云函数 JavaScript 语法正确
- TypeScript 类型检查通过
- CloudBase、安全、治理、后台和体验契约通过
- Mock 发布、审核、回复、举报、拉黑、通知和后台处理集成测试通过
