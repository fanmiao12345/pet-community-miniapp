# Changelog

## 1.1.0 - 2026-07-21

### Added

- 新增手机号强制授权登录页和隐私说明页
- 所有二维码入口在未验证手机号时统一跳转登录，并在授权后返回原始页面
- 云函数使用 `phonenumber.getPhoneNumber` 在服务端消费动态 code
- 手机号仅保存脱敏值和基于服务端密钥的 HMAC 摘要
- Mock 模式新增手机号授权模拟，便于游客模式演示
- 所有业务仓库调用和云函数增加手机号授权门禁
- 设置页展示脱敏手机号并支持账号注销清理绑定信息

### Security

- 禁止客户端直接提交手机号
- 同一手机号默认只能绑定一个未注销账号
- 公开用户接口不返回手机号摘要或完整手机号

## 1.0.1 - 2026-07-21

### Fixed

- 移除 `app.wxss` 中微信编译器不支持的通配选择器 `*`
- 将减少动画偏好规则精确限定到骨架屏 `.skeleton`
- 项目检查脚本新增 WXSS 通配选择器兼容性检查

## 1.0.0 - 2026-07-20

### Added

- CloudBase OpenID 用户体系、会话缓存和账号状态校验
- 云存储图片压缩、分类上传和失败清理
- 文本安全复检、图片异步审核和审核结果通知
- 关注流、评论回复、举报、拉黑和系统通知
- 管理员统计、举报处理、人工审核、用户禁言/停用和审计日志
- 隐私设置、账号注销、深色模式基础、骨架屏、错误重试和无障碍标签
- GitHub Actions、契约测试、Mock 集成测试、部署与发布文档

### Changed

- 所有身份判断从固定 Mock 用户升级为会话用户 ID
- 公共信息流仅展示审核通过且未被拉黑作者的内容
- 人工审核结果优先于迟到的异步审核回调
- 数据模型扩展至社区治理和运营后台
